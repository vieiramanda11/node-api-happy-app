<h1 align="center">hAPPy - API</h1>

> This project consists in build the API of an application that will allow users to search for orphanages and add new ones.

## Tools used

- Node.js
- Express
- Typeorm
- Yarn

## Install

```sh
yarn install
```

## Usage

```sh
yarn dev
```
